# Drenen

### Paquetes

Drenen utiliza:
* Python 2.7.11+
* Django 1.10


### Instalación y ejecución

Crear un entorno virtual (*Recomendado*)
```sh
$ virtualenv --no-site-packages env
$ source env/bin/activate
```

Instalar los paquetes
```sh
$ pip install -r requirements.txt
```
Correr el servidor
```sh
$ python manage.py runserver
```
